`keybindings.json`: [How to open keybindings.json](https://code.visualstudio.com/docs/getstarted/keybindings#_advanced-customization)

```js
// absolute path
{
	"key": "ctrl+shift+8",
	"command": "openFile",
	"args": "C:/warning.svg"
}
```
```js
// relative path
{
	"key": "ctrl+shift+8",
	"command": "openFile",
	"args": "./package.json"
}
```
